
## Galactic

A map of the Internet.

## Usage

    $ git clone https://github.com/SFDevLabs/galactic.git
    $ cd galactic
    $ npm install
    $ cp .env.example .env
    $ npm start

# Tests
Run test with:

	$ npm test
