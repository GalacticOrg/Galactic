Getting Started

Now you have the extension up and running. It's time to start making connections on the WikiWeb.

1
Create an account by logging in using the extension.
2
Get Connecting! Enable the extension bar via the button in the location bar.
----------

*Connection Types* _There are a few types of connections that exist within the WikiWeb:_ [[+] icon] *User Connections* Create a connection by clicking the [+] button and pasting a link. [ icon] *Page Links* These are existing links embeded into the page you are on. [backlink icon] *Backlinks* These are existing links from other sites to the page you are on. [Gears Icon] *WikiWeb Suggestions* Suggestions brought to you by WikiWeb's servers [warning: experimental]. -----------

*Privacy* _All connections in the WikiWeb are public. We do not yet offer private connections. [Public Icon] These connections are visible to everyone on the page itself and in our [a]public firehose[/a]. ----------- [^^ Icon] *Show/Hide All Connections* Use the [^^] Icon to toggle visibility of all connections to the page. Use [<^] to hide them. [Turn On/Off] Turn on the WikiWeb extension by clicking on our [W] logo your browser's navigations menu.

------------ Resources * Request features, report bugs and chat with our dev team via email. * If you need help, please don't hesitate to get in touch. * Visit our website to learn more about the WikiWeb.
