Getting Started

Now you have the extension up and running. It's time to start making connections on the WikiWeb.

1
Get Connecting! Enable the extension bar via the button in the location bar.
2
Create an account by logging in when using the extension.

Connection Types

There are a few types of connections that exist within the WikiWeb:

User Connections

Create a connection by clicking the button and pasting a link.

Page Links

These are existing links embeded into the page you are on by the author.

Backlinks

These are existing links from other sites to the page you are currently on.

WikiWeb Suggestions

Suggestions brought to you by WikiWeb's servers [experimental].

Privacy

All connections in the WikiWeb are public by default.

All connections are visible to everyone on the page itself and in our public firehose.

Show / Hide all Connections

Use the icon to toggle visibility of all connections to the page.
