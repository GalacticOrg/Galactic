About


WikiWeb is an Internet thesaurus that shows connections between similar content online. When you search on WikiWeb, by pasting in a URL, you're essentially asking what is like this?

Connections are crowdsourced by the WikiWeb community, which means behind each connection there is an explicit suggestion from a WikiWeb contributor that this is like that.

(This article is like that article, this podcast is like that essay, this video is like that book, etc.)

As the number of connections on the WikiWeb increases, so does the utility of WikiWeb as a discovery tool, so we hope you will join us in curating a connected web.

WikiWeb is a map of the Internet - a crowdsourced collection of connections between URLs that exists to better answer the question "what is like this?", online. The way it works is that contributors make connections between URL pairs which others can discover later. These URL pairs are called edges, and act as a pathway between similar ideas.

Our goal is to create a new way for people to discover information online, while at the same time creating an outlet for the synaptic burst that happens when the human brain does what it does best: make connections between things!

As the number of edges grows so does WikiWeb's usefulness, so we hope you will join us in connecting the Internet and making it better, together.
