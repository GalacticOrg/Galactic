About


Edgebase is a map of the Internet - a crowdsourced collection of connections between URLs that exists to better answer the question "what is like this?", online. The way it works is that contributors - contributors like you - make connections between URL pairs which others can discover later. These URL pairs are called edges, and act as a pathway between similar ideas.

Our goal is to create a new way for people to discover information online, while at the same time creating an outlet for the synaptic burst that happens when the human brain does what it does best: make connections between things!

As the number of edges grows so does the usefulness of Edgebase, so we hope you will join us in creating a better Internet, together.
